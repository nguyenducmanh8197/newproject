## 1. Design System

### 1.1. Color Palette

#### Primary Colors

```css
--primary-color: #1890ff; /* Ant Design Primary Blue */
--primary-hover: #40a9ff;
--primary-active: #096dd9;
--primary-light: #e6f7ff;
```

```css

--warning: #faad14; /* Budget warning (80%) */

```

#### Neutral Colors

```css
--text-primary: #262626;
--text-secondary: #8c8c8c;
--text-disabled: #bfbfbf;
--background: #ffffff;
--background-light: #fafafa;
```


```css
/* Font Family */
--font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif;

--font-size-xs: 12px;
--font-size-sm: 14px;
--font-size-base: 16px;
--font-size-lg: 18px;
--font-size-xl: 20px;
--font-size-xxl: 24px;

/* Font Weights */
--font-weight-normal: 400;
--font-weight-medium: 500;

--font-weight-bold: 700;

/* Line Heights */
--line-height-tight: 1.2;
--line-height-normal: 1.5;
--line-height-relaxed: 1.75;
```

```css
--spacing-xs: 4px;
--spacing-sm: 8px;
--spacing-md: 16px;
--spacing-lg: 24px;
--spacing-xl: 32px;
--spacing-xxl: 48px;
```

### 1.4. Border Radius

```css
--radius-sm: 4px;
--radius-md: 8px;
--radius-lg: 12px;
--radius-circle: 50%;
```

### 1.5. Shadows

```css
--shadow-sm: 0 2px 4px rgba(0, 0, 0, 0.08);
--shadow-md: 0 4px 12px rgba(0, 0, 0, 0.12);
```
